# Guard::Herbalizer

Guard plugin to convert HAML to ERB using [herbalizer](https://github.com/danchoi/herbalizer)

## Installation

Add this line to your application's Gemfile:

    gem 'guard-herbalizer'

And then execute:

    $ bundle
    $ bundle exec guard init
    
And edit your `Guardfile`
